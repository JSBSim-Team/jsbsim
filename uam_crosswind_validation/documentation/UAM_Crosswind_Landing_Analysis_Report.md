## 1. 연구 개요

### 1.1 연구 배경
도심항공교통(Urban Air Mobility, UAM)은 차세대 교통 시스템으로 주목받고 있으나, 도심 환경의 복잡한 바람 조건에서의 안전한 착륙이 중요한 과제입니다. 특히 횡풍 조건에서의 착륙 시 발생하는 좌우 편차는 착륙장 안전성과 직결되는 핵심 요소입니다.

### 1.2 연구 목적
본 연구는 UAM 기체(멀티로터 타입)의 횡풍 조건에서 착륙 시 발생하는 좌우 편차를 정량적으로 분석하고, 안전 운항 기준을 제시하는 것을 목적으로 합니다.

### 1.3 연구 방법
- 6자유도 비선형 비행동역학 모델링
- JSBSim 기반 시뮬레이션 환경 구축
- 다양한 횡풍 조건에 대한 Monte Carlo 시뮬레이션
- 통계적 분석 및 임계 조건 도출

## 2. 이론적 배경 및 수학적 모델

### 2.1 UAM 기체 모델
본 연구에서는 멀티로터 타입의 UAM 기체를 대상으로 하며, 주요 제원은 다음과 같습니다:

**기체 제원**
- 최대 이륙중량: 2,500 lbs (1,134 kg)
- 로터 간격: 6.0 m
- 전체 길이: 4.57 m (15 ft)
- 관성모멘트: Ixx = Iyy = 700 kg⋅m², Izz = 1,166 kg⋅m²

### 2.2 횡방향 동역학 방정식

UAM 기체의 횡방향 운동은 다음 미분방정식으로 기술됩니다:

#### 2.2.1 병진 운동 방정식
```
dv/dt = (Y/m) - ru + pw
```

여기서:
- `v`: 측방향 속도 [m/s]
- `Y`: 측방향 공기역학적 힘 [N]
- `m`: 기체 질량 [kg]
- `r`, `p`: 요각속도, 롤각속도 [rad/s]
- `w`: 수직방향 속도 [m/s]

#### 2.2.2 회전 운동 방정식
```
dp/dt = (L + (Iyy - Izz)qr)/Ixx
dr/dt = (N + (Ixx - Iyy)pq)/Izz
```

여기서:
- `L`: 롤 모멘트 [N⋅m]
- `N`: 요 모멘트 [N⋅m]
- `q`: 피치각속도 [rad/s]
- `Ixx`, `Iyy`, `Izz`: 관성모멘트 [kg⋅m²]

#### 2.2.3 자세각 변화 방정식
```
dφ/dt = p + q⋅sin(φ)⋅tan(θ) + r⋅cos(φ)⋅tan(θ)
dψ/dt = (q⋅sin(φ) + r⋅cos(φ))/cos(θ)
```

여기서:
- `φ`: 롤각 [rad]
- `ψ`: 요각 [rad]  
- `θ`: 피치각 [rad]

### 2.3 공기역학적 힘과 모멘트

#### 2.3.1 측방향 힘 (Side Force)
```
Y = q̄⋅S⋅[Cy_β⋅β + Cy_δr⋅δr + Cy_p⋅p̂ + Cy_r⋅r̂]
```

#### 2.3.2 롤 모멘트 (Rolling Moment)
```
L = q̄⋅S⋅b⋅[Cl_β⋅β + Cl_p⋅p̂ + Cl_r⋅r̂ + Cl_δa⋅δa + Cl_δr⋅δr]
```

#### 2.3.3 요 모멘트 (Yawing Moment)
```
N = q̄⋅S⋅b⋅[Cn_β⋅β + Cn_r⋅r̂ + Cn_p⋅p̂ + Cn_δr⋅δr + Cn_δa⋅δa]
```

여기서:
- `q̄ = ½ρV²`: 동압 [N/m²]
- `S`: 기준면적 [m²]
- `b`: 기준 길이 (로터 간격) [m]
- `β`: 사이드슬립각 [rad]
- `p̂, r̂`: 무차원 각속도
- `δa, δr`: 에일러론, 러더 편향각 [rad]

### 2.4 횡풍 영향 모델링

#### 2.4.1 상대속도 계산
```
u_rel = u - Vwind⋅cos(ψwind)
v_rel = v - Vwind⋅sin(ψwind)
Vrel = √(u_rel² + v_rel² + w_rel²)
```

#### 2.4.2 사이드슬립각 계산
```
β = arcsin(v_rel/Vrel)
```

여기서:
- `Vwind`: 풍속 [m/s]
- `ψwind`: 풍향 [rad] (0° = 정면풍, 90° = 우측풍)

### 2.5 공기역학 계수

UAM 기체에 적용된 주요 공기역학 계수는 다음과 같습니다:

| 계수 | 값 | 물리적 의미 |
|------|----|-----------| 
| Cy_β | -0.25 | 사이드슬립에 의한 측력 |
| Cy_δr | -0.12 | 러더에 의한 측력 |
| Cy_p | 0.02 | 롤레이트에 의한 측력 |
| Cy_r | 0.08 | 요레이트에 의한 측력 |
| Cl_β | -0.08 | 사이드슬립에 의한 롤모멘트 (상반각 효과) |
| Cl_p | -0.45 | 롤 댐핑 계수 |
| Cl_r | 0.02 | 요레이트에 의한 롤모멘트 |
| Cl_δa | 0.15 | 에일러론 효과 |
| Cl_δr | 0.01 | 러더에 의한 롤모멘트 |
| Cn_β | 0.12 | 방향안정성 계수 |
| Cn_r | -0.25 | 요 댐핑 계수 |
| Cn_p | -0.01 | 롤-요 커플링 |
| Cn_δr | -0.08 | 러더 효과 |
| Cn_δa | -0.005 | 에일러론에 의한 adverse yaw |

## 3. 시뮬레이션 설정 및 방법론

### 3.1 시뮬레이션 환경
- **시뮬레이션 도구**: JSBSim + Python 통합 환경
- **수치해석 방법**: 4차 Runge-Kutta 방법
- **시간 간격**: 0.05초
- **시뮬레이션 시간**: 30초 (착륙 접근 단계)

### 3.2 초기 조건
```
위치: x=0, y=0, z=100m (접근 고도)
속도: u=20m/s (접근속도), v=0, w=-2m/s (하강속도)
자세: φ=0°, θ=-3° (접근각), ψ=0°
각속도: p=q=r=0 rad/s
```

### 3.3 시뮬레이션 조건
- **풍속**: 2, 5, 8, 10, 12, 15 m/s (6단계)
- **풍향**: 0°, 30°, 60°, 90°, 120°, 150°, 180° (7단계)
- **총 케이스 수**: 42개 조합
- **제어시스템**: PID 자동제어 (롤/요 안정화)

### 3.4 제어 법칙
```python
# 롤각 제어 (수평 유지)
δa = Kp_φ⋅φ + Kd_φ⋅p
Kp_φ = -2.0, Kd_φ = -0.5

# 요각 제어 (방향 유지)  
δr = Kp_ψ⋅ψ + Kd_ψ⋅r
Kp_ψ = -1.0, Kd_ψ = -0.3
```

## 5. 상세 분석

### 5.1 물리적 해석

#### 5.1.1 풍향별 특성
- **정면풍 (0°)**: 측방편차 없음 (예상대로)
- **측풍 (90°)**: 직접적인 측방력으로 인한 최대 편차
- **사풍 (120°, 150°)**: 측방력 + 요모멘트 결합으로 인한 최대 편차
- **후방풍 (180°)**: 상대적으로 작은 편차 (접근속도 증가 효과)

#### 5.1.2 풍속 의존성
대부분의 풍향에서 풍속과 측방편차는 거의 선형적 관계를 보입니다:
```
측방편차 ≈ α × 풍속 + β
```

### 5.2 제어시스템 분석
현재 PID 제어시스템의 한계:
1. **롤 제어**: 비교적 효과적 (최대 롤각 <5°)
2. **요 제어**: 횡풍 조건에서 한계 노출
3. **적분 제어 부재**: 정상상태 오차 존재

### 5.3 공기역학적 특성
- **방향안정성 (Cn_β = 0.12)**: 양수로 기본적인 안정성 확보
- **롤-사이드슬립 커플링 (Cl_β = -0.08)**: 상반각 효과로 복원력 제공
- **댐핑 특성**: 롤 댐핑(-0.45), 요 댐핑(-0.25)으로 진동 억제

## 6. 결론 및 권고사항

### 6.1 주요 결론

1. **횡풍 민감도**: UAM 기체는 횡풍에 매우 민감하며, 특히 120°-150° 사풍에서 최대 편차 발생

2. **안전성 우려**: 현재 설계로는 중간 이상의 횡풍 조건에서 안전 착륙이 어려움 (안전률 14.3%)

3. **임계 풍속**: 10 m/s 이상의 횡풍에서는 200m 이상의 측방편차 발생 가능

4. **풍향별 차이**: 정면풍 대비 측풍에서 15배 이상의 편차 증가

### 6.2 개선 방안

#### 6.2.1 제어시스템 개선
1. **적응제어 적용**:
   ```
   δr(t) = Kp_ψ⋅ψ + Ki_ψ⋅∫ψdt + Kd_ψ⋅ψ̇ + Kff⋅β_est
   ```

2. **풍속 추정 및 전향보상**:
   ```
   β_est = arctan(Vwind⋅sin(ψwind)/Vapproach)
   ```

3. **모델 예측 제어 (MPC)** 도입

#### 6.2.2 기체 설계 개선
1. **수직꼬리날개 추가**: Cn_β 증가 (0.12 → 0.20)
2. **로터 배치 최적화**: 요 제어 모멘트 증대
3. **랜딩기어 개선**: 지상 활주 안정성 향상

#### 6.2.3 운항 기준
1. **횡풍 한계**: 8 m/s 이하로 제한
2. **접근각 조정**: 풍향에 따른 crab angle 적용
3. **착륙장 폭 확대**: 최소 50m 이상 권장

### 6.3 추가 연구 필요 사항

1. **난류 영향 분석**: 도심 환경의 복잡한 난류 특성 반영
2. **파일럿 모델링**: 수동 조종 시 인간 요소 고려
3. **다양한 기체 형상**: 틸트로터, 복합형 UAM 분석
4. **실제 비행시험**: 시뮬레이션 모델 검증

## 7. 수학적 부록

### 7.1 무차원 파라미터
```
p̂ = pb/(2V)  : 무차원 롤각속도
r̂ = rb/(2V)  : 무차원 요각속도
Re = ρVc/μ   : 레이놀즈 수
```

### 7.2 안정성 판별 기준
```
Cn_β > 0    : 방향 안정성
Cl_β < 0    : 상반각 안정성 (UAM의 경우)
Cn_r < 0    : 요 댐핑
Cl_p < 0    : 롤 댐핑
```

### 7.3 제어 효율성
```
δa_eff = Cl_δa / Cn_δa = 0.15 / (-0.005) = -30
δr_eff = Cn_δr / Cl_δr = -0.08 / 0.01 = -8
```
