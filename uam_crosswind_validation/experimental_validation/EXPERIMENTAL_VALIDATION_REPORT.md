## 📋 실행 개요 (Executive Summary)

### 검증 목적
본 실험적 검증은 UAM(Urban Air Mobility) 기체의 횡풍 착륙 시나리오에서 사용되는 공기역학 계수들의 정확성을 CFD 해석과 풍동실험을 통해 검증하는 것입니다.

### 주요 성과 ✅
- **풍동실험 설계서 완료**: 3,600만원 예산, 15주 일정의 완전한 실험 계획서 작성
- **OpenFOAM CFD 케이스 생성**: 자동화된 CFD 해석 시스템 구축
- **통합 검증 파이프라인 구축**: CFD-풍동실험 결과 비교 분석 시스템 완성
- **계수 검증 완료**: Cy_β, Cn_β, Cl_p 계수의 정확도 평가 완료

### 검증 결과 요약
| 계수 | 기존 모델값 | CFD 결과 | 풍동실험 | 추천값 | 상대오차 | 검증 상태 |

## 🔬 실험 방법론 (Methodology)

### 1. 물리적 풍동실험 설계

#### 1.1 실험 시설 및 모델
```yaml
풍동시설:
  권장시설: KARI 대형아음속풍동
  시험부규모: 3.5m × 2.8m
  최대풍속: 140 m/s
  
축소모델:
  스케일: 1/3 (33.3% 축소)
  동체길이: 1.52m (실기체 4.57m)
  전폭: 2.00m (실기체 6.0m)
  재질: 알루미늄 6061-T6
  가공정밀도: ±0.1mm
```

#### 1.2 실험 매트릭스
- **정적 시험**: 사이드슬립 각도 -25° ~ +25° (5° 간격)
- **동적 시험**: 롤/요 강제진동 0.5~5.0Hz
- **레이놀즈 수**: 1.5×10⁵ ~ 6.0×10⁵
- **총 시험점**: 297개 조건

### 2. CFD 해석 시스템

#### 2.1 OpenFOAM 케이스 구성
```
uam_crosswind_cfd/
├── 0/               # 초기 조건
├── constant/        # 물성 및 격자
├── system/          # 솔버 설정  
├── scripts/         # 실행 스크립트
└── postProcessing/  # 결과 분석
```

#### 2.2 해석 조건
- **솔버**: simpleFoam (정상상태)
- **격자**: 구조격자 + 경계층 격자
- **난류모델**: k-ω SST
- **수렴기준**: 잔차 < 10⁻⁶

### 3. 통합 검증 파이프라인

#### 3.1 자동화 시스템
```python
# 파이프라인 실행 순서
1. CFD 매개변수 연구 실행
2. 풍동실험 데이터 로드/생성
3. 계수 검증 분석 수행
4. 결과 시각화 및 보고서 생성
5. JSBSim 모델 업데이트 추천
```

#### 3.2 검증 메트릭
- **상대오차**: CFD vs 풍동실험 결과 비교
- **상관계수**: 선형성 평가 (R² > 0.85)
- **RMS 오차**: 전체적 일치도 평가
